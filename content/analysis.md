# Stratford PAL — Site Analysis

## Current Site Assessment (stratfordpal.org)

---

### Strengths
- Clear mission and core messaging ("Inspiring Positive Change")
- Good foundational content: programs listed with descriptions
- Contact info present (phone, email, address)
- Active social media presence (Facebook, Instagram)
- Donation link in place (Donorbox)
- Volunteer call-to-action
- Calendar page exists
- Partners acknowledged

---

### Weaknesses

**Visual Design**
- Dated Wix template with minimal customization
- Inconsistent typography and spacing
- No visual hierarchy to guide users through content
- Lack of imagery sections and program photos being used well
- No consistent color system or brand language

**Navigation**
- Navigation links not clearly visible in extracted content
- No clear page structure for most subpages (events, contact, scholarships 404)
- Multiple pages return 404 errors (contact, events/calendar detail, scholarships, get-involved)

**Content Structure**
- Homepage text is a wall of words with no scannable structure
- Programs listed but without formatting, visuals, or clear hierarchy
- No dedicated scholarships page (404)
- No dedicated get-involved page (404)
- Contact page returns 404
- Events/calendar is nearly empty ("Please see programs for registration details")

**UX Issues**
- No mobile-optimized layout evident from structure
- Single long scroll with no sectioning
- No card layouts — all text blocks
- No prominent CTAs throughout pages (only "Join Us" email link on homepage)
- No program filtering or categorization

**Messaging Issues**
- Headline is decent but not immediately action-oriented
- No clear answer to "What is PAL?" above the fold
- Programs buried, not featured
- No stats or social proof
- No family-facing language (primarily officer/org voice)
- Audience segments (parents, donors, volunteers) not addressed distinctly

**Conversion Gaps**
- Only one CTA on homepage (email link)
- No registration flow
- No dedicated donate page path (link in nav only)
- No scholarship inquiry path
- No volunteer application flow (link hidden)
- No newsletter/notification signup
- No "upcoming events" with registration links

---

### Opportunities

1. **Build missing pages** — Contact, Events, Scholarships, Get Involved are all 404s
2. **Restructure homepage** into clear sections (Hero → Mission → Programs → Events → Scholarships → Get Involved → CTA)
3. **Add program cards** with categories, schedules, age groups, and registration CTAs
4. **Create scholarship page** highlighting accessibility and financial assistance
5. **Build volunteer/donation funnel** with clear steps
6. **Add event notification signup** for summer program registration windows
7. **Strengthen mobile experience** with a mobile-first layout
8. **Add social proof** — statistics, community impact numbers
9. **Improve nav** with clear page structure

---

### Design Direction
- Modern, clean nonprofit aesthetic (not corporate, not overcrowded)
- Navy + Gold color palette (trust + community energy)
- Card-based program layouts
- Strong typography hierarchy
- Warm, community-centered tone
- Mobile-first responsive design
